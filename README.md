This project is aimed at building a technology trading application such as computers, laptops, phones ... 
It provides features such as product filtration, shopping cart and orders and management parts for admin.
<h2>🛠️ Technologies Used</h2>
<ul>
  <li>Frontend: React.js, Redux, Tailwind CSS, Axios, Recharts</li>
  <li>Backend: Spring Boot, Spring Security, Spring Data JPA</li>
  <li>Database: MySQL</li>
  <li>Authentication: JWT (JSON Web Tokens)</li>
  <li>Payment Gateway: Integration payment with VNPay</li>
  <li>Build Tools: Maven, Webpack</li>
</ul>
<h2>Frontend</h2>

- ✅ Set up React project structure 
- ✅ Implement a Home Page, Store and Contact
- ✅ Pages Navigation & Categories Page with Filters
- ✅ Product Detail Page
- ✅ Chat support with admin
- ✅ Shopping Cart Functionality
- ✅ User Authentication (sign-up, login, logout)
- ✅ Checkout Process

<h2>Backend</h2>

- ✅ Set up Spring Boot project
- ✅ Implement basic product API
- ✅ User authentication and authorization with JWT
- ✅ Order processing and management
- ✅ Chat realtime with websocket
- ✅ Integration with payment gateway
- ✅ Admin dashboard for product/order management
